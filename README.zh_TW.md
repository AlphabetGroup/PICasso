# <img src="images/app_logo.png" width="48" height="48" valign="middle"/> PICasso 拼圖藝術 (PICasso Collage)

**PICasso** 是一款創意拼圖與高級圖片編輯應用程式。它汲取了畢卡索式的藝術靈感，將靈動的物理交互動畫、優雅的毛玻璃視覺元素和全套的專業修圖工具完美融合，帶來絲滑的 60/120fps 創作體驗。

## ✨ 核心特性

### 1. 靈動交互與吸頂玻璃標題欄
* **不對稱瀑布流卡片**：採用優雅的色彩漸變設計，完美契合現代設計美學。
* **物理觸感回饋 (`_BouncyCard`)**：卡片擁有彈性的物理回饋——按下時縮放至 `0.94` 倍，釋放時以回彈曲線 (`Curves.easeOutBack`) 靈動復位。
* **展開式路由過渡動畫**：自定義過渡路由融合了漸顯、縮放和微過衝回彈，呈現出極具視覺衝擊力的卡片展開動效。
* **滾動驅動的毛玻璃吸頂欄**：隨滑動距離動態調節標題高度（`110` 至 `60`）、文字大小、副標題漸隱，並柔和過渡出高斯模糊（`sigma 10`）與半透明邊界線。

### 2. 專業級圖片編輯器 (P圖)
* **無裁剪裁剪框**：即時角落調整手勢柄可以超出畫布邊緣，確保即使在邊界座標處也易於拖拽。支持固定比例鎖定（`自由`、`1:1`、`4:3`、`3:4`、`16:9`、`9:16`）。
* **自適應旋轉視口**：在圖片進行左右 90 度旋轉時，自動計算並以 `SizedBox` 顯式設置未旋轉前的容器尺寸。這避免了佈局拉伸導致的比例擠壓，確保長圖/寬圖以最大尺寸清晰呈現。
* **所見即所得雙擊行內文本編輯**：雙擊文本圖層直接在畫布上修改內容。支持最小佈局寬度、光標記憶以及避开軟鍵盤遮擋。
* **零延遲手勢操作**：單指拖拽流暢移動貼紙和文本圖層，雙指捏合輕鬆進行無延遲縮放及旋轉。
* **15 款質感濾鏡**：內置高保真濾鏡（包括懷舊、Lomo、褐色、鮮豔、粉嫩、戲劇、冷/暖色等），即時呈現即時濾鏡效果。
* **高精度色彩調色**：滑動桿即可高精度調節亮度、對比度和曝光參數。
* **自由塗鴉與混合橡皮擦**：可調節粗細與顏色的畫筆工具，橡皮擦實現物理擦除，不破壞底部基底圖層。
* **智能美顏**：提供即時溫和的皮膚美白與高斯模糊滑膚功能。

### 3. 模板拼圖
* **自適應分鏡排版**：針對 1 ~ 9 張不同數量的圖片，自動提供不同方案的經典與斜切拼鏡。
* **手勢控制**：每個格槽獨立支持雙指縮放、拖拽移動與位置微調，且默認以 `BoxFit.contain` 保持圖片原始比例，避免畫面被切割。
* **間距與圓角調整**：支持動態滑動調節分格的間距粗細與圓角弧度。
* **位置快速交換**：長按並拖拽分格圖片即可與任意分格進行無縫位置對調。

### 4. 自由拼貼
* **零限制創作**：自由拖拽、單/多點捏合縮放及旋轉畫布上的圖片，帶有物理回彈和絲滑的爆裂散開初始動畫。
* **井字形基準線**：在拖拽圖片時，畫布上會漸顯出“井”字形三分線，輔助進行黃金分割排版。
* **快速對齊預設**：一鍵執行**網格 (Grid)**、**散落 (Scattered)**、**環繞 (Circular)**、**堆疊 (Stacked)** 四種快速佈局。
* **沉浸折疊選單**：控制欄可收起折疊，最大化騰出創作區域。

### 5. Comic 漫畫分鏡
* **動感斜切佈局**：利用自定義分鏡演算法裁剪圖片，生成衝擊力極強的動感漫畫格子，默認以 `BoxFit.contain` 完整呈現圖片。
* **網格預覽選擇器**：用迷你線框圖方式呈現佈局縮略圖，一目了然。
* **拖拽對調**：支持在不規則斜切格之間拖拽圖片交換位置，並保留原裁剪狀態。

### 6. 濾鏡相機
* **水平輔助參考線**：當傾斜角對準在安全角度內時，水平儀變綠並提供穩定回饋。
* **前後雙攝 PIP (畫中畫) 連拍**：有效解決大部分手機攝像頭硬件搶佔衝突。拍攝時順序讀取前置攝像頭圖像並釋放，隨後觸發主攝像頭成像合成。
* **无干扰通知 toast**：將傳統的保存成功彈窗替換為懸浮玻璃態通知，靜默保存，極佳的沉浸式體驗。

### 7. 長圖拼接與九宮格切图
* **長圖拼接**：拼接多至 20 張圖片，支持橫向或豎向排列。可在縮略圖欄長按拖拽排序，點擊段落可對單張圖進行旋轉、鏡像與刪除。
* **九宮格切圖**：支持 2x2, 3x3, 3x1, 1x3 切圖，切圖後自動輸出帶編號的切割圖集至本地。

## 🎨 主題與語言自定義

PICasso 從底層構建了對多語言本地化和自適應顯示首選項的動態支持：

### 1. 自適應深色與淺色模式
* **動態樣式變換**：在雅致的炭黑深色模式（`Color(0xFF121214)`）與高對比度清爽的淺色模式（`Color(0xFFF5F5F7)`）之間無縫切換。
* **即時廣播**：基於全局 `AppState` 狀態機。當用戶切換淺色、深色或系統默認主題時，更改將即時廣播至所有頁面，無需重新啟動應用。
* **智能對比度調節**：畫布邊框、文本輸入框、貼紙控制手勢柄、輔助線以及設置選單等，會動態調整色彩以確保最佳的可讀性。

### 2. 多語言（本地化）支持
* **系統語言智能檢測**：在應用啟動時自動讀取設備的系統語言（例如腳本字元與國家/地區代碼），並設定最佳的初始語言。
* **語言熱切換**：支持以下三種語言的無縫熱切換：
  - **简体中文**
  - **繁體中文**
  - **English**
* **本地化翻譯架構**：利用應用內字典映射關係，通過 `AppState.instance.translate('key')` 讀取對應的本地化字串。所有頁面均監聽語言變動，自適應刷新文字展示。
